# librosa-practice

오디오 데이터가 실제로 어떻게 분석되고 시각화하는지 궁금해, 소리를 눈으로 보는 실습을 진행.

 ### librosa 라이브러리를 활용하여 오디오 데이터를 분석하고, 파형과 Mel-Spectrogram을 시각화  

librosa를 이용해 오디오 데이터를 불러오고, 그 안에 담긴 소리를 두 가지 방식으로 시각화.

상단 그래프 (파형)
→ 시간에 따른 소리의 진폭 변화를 나타냄  
-> 소리가 언제 커지고 작아지는지 볼 수 있음


하단 그래프 (Mel-Spectrogram)
→ 시간에 따른 주파수 변화 및 소리의 강도를 색으로 표현  
-> 어떤 음이 언제 강하게 들리는지 볼 수 있음
